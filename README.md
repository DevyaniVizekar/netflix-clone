# Netflix Clone

A responsive Netflix homepage clone built using HTML and CSS.  
This project focuses on recreating the layout, design, and visual experience of a real-world streaming platform.

## Overview
This project is a detailed frontend clone of Netflix’s landing page. It replicates the structure, styling, and responsiveness of the original website to provide a realistic user interface experience.

## Features
- Netflix-style homepage layout
- Fully responsive design
- Navigation bar with logo and buttons
- Hero section with background image and call-to-action
- Multiple content sections with images and text
- Footer with structured links
- Clean and visually appealing UI

## Tech Stack
- HTML
- CSS

## Key Learnings
- Building complex and large-scale layouts
- Advanced CSS styling and positioning
- Working with Flexbox for alignment and structure
- Creating responsive designs for different screen sizes
- Structuring real-world UI components

## Challenges Faced
- Replicating the exact layout and spacing
- Managing responsiveness across devices
- Aligning multiple sections consistently
- Handling background images and overlays

## Project Structure
- index.html
- style.css
- assets/ (images and media files)

## Project Status
This is a frontend-only clone and does not include backend functionality or video streaming features.
